# Changelog

## 0.1.0 (2026-05-14)

- Initial release
- `set_style()` — one-line Nature/Science/IEEE journal styling
- `make_fig()` / `savefig()` — journal-dimensioned figures
- `get_cmap()` — CMasher + Crameri + Matplotlib colour maps in one call
- `polish_axes()` — bottom-left spines, inward ticks
- Plot templates: `plot_curves()`, `plot_field()`, `plot_field_comparison()`,
  `plot_error_map()`, `plot_training_curves()`
- `demo()` — full gallery generation
